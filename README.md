# 剧本杀小馆 🕵️

局域网联机的网页剧本杀游戏。本电脑开机运行服务器，朋友的手机/平板/电脑连同一个 WiFi，用浏览器打开网址就能一起玩。**无需安装任何依赖**（纯 Python 标准库）。

## 怎么开始

1. 双击 `启动游戏.bat`（或命令行运行 `python server.py`）
2. 窗口里会显示两个地址：
   - 本机玩：`http://localhost:8899`
   - 朋友玩：`http://<你的局域网IP>:8899`（窗口里会直接打印出来）
3. 每个人在自己设备的浏览器打开地址，输入昵称
4. 一个人选剧本创建房间，把 **4 位房号** 告诉其他人，其他人输入房号加入

> ⚠️ 第一次运行时 Windows 可能弹出防火墙提示，请勾选「专用网络」并点击**允许访问**，否则其他设备连不上。
> 如果没弹窗但别人连不上，用管理员权限运行一次：
> `netsh advfirewall firewall add rule name="jubensha" dir=in action=allow protocol=TCP localport=8899`

## 游戏流程

1. **大厅**：等人齐（4人），每人自选角色或点「随机分配」
2. **阅读剧本**：每人只能看到自己角色的故事、秘密和任务（别给别人看手机！）
3. **搜证轮**（两轮）：每人每轮 2 次搜证机会，去各地点找线索；搜到的线索可以**立即公开**或**先私藏**（私藏的之后随时可公开）
4. **讨论**：全程可用「讨论」页聊天，也可以直接当面聊
5. **投票**：每人指认心中的凶手（可改票）
6. **真相揭晓**：公布凶手、投票结果、完整复盘和每个人的秘密

各阶段由**房主**点按钮推进。结束后房主可点「重开」回到大厅再来一局。

## 自带剧本

| 剧本 | 人数 | 风格 | 难度 |
|---|---|---|---|
| 《午夜美术馆》 | 4人 | 现代悬疑：赝品、遗嘱与染血铜像 | 中等 |
| 《雾隐客栈》 | 4人 | 古风武侠：大雪封山的复仇局 | 中等偏上 |
| 《寿宴上的窃案》 | 4人 | 欢乐轻松：无人死亡的失窃案，适合新手 | 简单 |
| 《午夜电台》 | 5人 | 现代硬核：录播制造的完美不在场证明 | 中等偏硬核 |
| 《宗门大比前夜》 | 5人 | 欢乐修仙：筑基丹失窃闹剧，无人死亡 | 简单欢乐 |

房间人数由剧本角色数自动决定：选 4 人本就是 4 人房，5 人本就是 5 人房。

## 添加新剧本

在 `scripts/` 文件夹里参照现有 JSON 格式新建一个文件，重启服务器即可。关键字段：

- `characters`：角色列表，每个角色有公开介绍 `public` 和私密的 `story` / `secrets` / `goals`
- `murderer`：凶手的角色 id
- `rounds`：搜证轮，每轮若干地点，每个地点若干线索
- `search_points`：每轮每人搜证次数（默认 2；建议线索总数 = 人数 × 次数）
- `truth`：结尾复盘长文

## 技术说明

- 纯 Python 标准库 `http.server`，端口 8899（可在 `server.py` 顶部改 `PORT`）
- 房间状态存内存，重启服务器房间会清空；6 小时无人访问的房间自动清理
- 前端 1.5 秒轮询同步，刷新页面/断线可自动重连回房间
