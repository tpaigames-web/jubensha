# 密封内容警告

以下路径是**剧本正文**，属于密封内容：

```
cloud/content/            创作中间产物（真相层、角色本、线索、碎片、答案、播报、复盘）
cloud/scripts/*/content.pack   打包后的文案包
cloud/tools/build-*.mjs        引擎自带剧本的正文生成器
cloud/tools/enrich/*.mjs       旧剧本的增补层（含角色关系与秘密）
```

**任何人不得打开这些文件——包括未来接手的 AI 助手。**

委托人本人要作为玩家游玩本作。被剧透会直接毁掉这个项目的意义。

唯一可以公开阅读的产物是 `content/QC-report.md`，它只含统计与校验结果，不含任何剧情。

需要改动剧本时，改生成器/中间产物并重新打包，**不要在对话里引用正文内容**。
